# Changelog

## 1.0.0

- **Initial Open Source Release**: The `Niskala Model Gen` project has been fully sanitized and refactored for public release.
- **Features**:
    - Complete AST parsing of OData v4 JSON metadata XML.
    - Automatic model, class, and testing script generation with high-coverage compliance.
    - Flexible enumeration overrides string allocations.
    - Built-in `dart fix --apply` automated integration for 100% linter adherence (`very_good_analysis`).
